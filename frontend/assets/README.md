# Assets Directory

Esta carpeta contendrá los recursos visuales de la aplicación:

## Estructura

- `images/` - Logos, iconos, imágenes estáticas
- `animations/` - Archivos Lottie (.json)
- `icons/` - Iconos personalizados
- `fonts/` - Fuentes personalizadas (Poppins)
